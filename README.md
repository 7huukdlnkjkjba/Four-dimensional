# 爬取博时黄金C数据.py独断万古：量子高维分析与精准预测系统 v8.0

这是一个融合**金融数据量化**、**量子纠缠理论**、**传统易经占卜**与**前沿机器学习**的技术性 Python 脚本，用于对博时黄金C（基金代码 002611）进行**过去、现在、未来三重维度**的全方位“高维算命”式分析，并精准预测次日涨跌幅。系统通过爬取基金净值、黄金新闻，结合六爻起卦、新闻情感分析、量子纠缠风险指数，以及基于随机森林、XGBoost、LSTM 的集成学习模型，最终输出综合投资建议与精确数值预测——**独断万古，洞见黄金未来**。

## ✨ 核心功能特性

### ⏳ 时间维度融合（新增独断万古特性）
- **回溯过去因果**：基于历史净值、技术指标、新闻事件，构建时间序列因果图，识别影响黄金走势的关键历史节点。
- **洞察现在状态**：实时获取最新市场数据、新闻情感、量子纠缠风险，捕捉当前市场情绪与结构。
- **前瞻未来趋势**：通过多模型集成学习，结合时间序列预测、概率推断，输出次日涨跌幅的精准预测及置信区间。
- **独断万古**：系统将过去数据、现在状态、未来预测进行高维统合，形成超越单一时间维度的全局判断，实现“独断万古”的决策能力。

### 🔮 量子高维分析（原 v6.0 保留）
- **实时基金数据**：获取博时黄金C最新净值和日涨跌幅（数据源：天天基金网 API）
- **实时黄金新闻**：从新浪财经 RSS 和百度新闻抓取黄金相关标题
- **量子纠缠风险分析**：基于多只关联基金（黄金、白银、原油、纳指等）的历史收益率构建量子态密度矩阵，计算冯·诺依曼熵、量子互信息及纠缠风险指数，映射为市场结构风险得分
- **六爻占卜**：模拟投掷三枚硬币（或基于黄金 ETF 成交量数据）起卦，生成六爻本卦和变卦
- **卦象解析**：将卦象映射为 64 卦名称，并提供针对黄金投资的解释和吉凶评分
- **新闻情感分析**：基于内置关键词词典对新闻标题进行多空打分
- **综合建议（初级版）**：融合卦象吉凶、量子风险得分、情感得分和当日涨跌幅，给出从“强烈看多”到“快跑”的结论

### 🤖 机器学习精准预测（新增 v8.0）
- **技术指标计算**：自动计算 MA5/10/20/60、RSI、MACD、布林带、动量、波动率、成交量比率等 12 个技术指标
- **多模型集成学习**：
  - 随机森林 (Random Forest)
  - XGBoost（可选，需安装 xgboost）
  - LSTM 神经网络（可选，需安装 tensorflow）
- **集成预测**：对三个模型的预测结果取平均，输出**次日精准涨跌幅**及**预测净值**
- **95% 置信区间**：基于各模型预测的方差估算区间
- **特征重要性分析**：输出对预测最重要的 5 个特征

### 📊 实盘验证与优化（新增 v8.0）
- **自动记录预测**：每次预测结果（日期、集成预测值、各模型预测值）自动保存至 `predictions_log.csv`
- **手动更新实际值**：次日可手动录入实际涨跌幅，完善历史记录
- **统计评估**：随时查看历史预测的**方向准确率**、平均绝对误差 (MAE)、均方根误差 (RMSE)、**盈亏比**及累计收益
- **持续优化**：可根据实盘表现调整模型权重或添加新特征

2. **运行脚本**：
   ```bash

   ```
3. **交互菜单**：
   ```
   博时黄金C 多模型集成预测系统
   1. 运行今日预测并记录
   2. 查看历史统计

   请选择操作 (1/2/3):
   ```
   - 选择 `1` 将执行完整分析（量子高维分析 + 技术指标 + 集成预测），并自动保存预测结果。
   - 选择 `2` 可查看历史预测准确率、盈亏比等统计。

## precise_prediction_fixed.pyAI量化投资脚本介绍

这是一个基于机器学习的量化投资工具，专门用于预测**博时黄金C（基金代码：002611）** 的次日涨跌幅，并提供历史回测评估功能。脚本集成了多个机器学习模型（线性回归、随机森林、XGBoost、LSTM），通过加权集成的方式输出预测结果，并自动记录预测与实际表现，帮助投资者验证策略有效性。

### ✨ 主要功能

1. **多模型集成预测**  
   - 使用 **线性回归 (LR)**、**随机森林 (RF)**、**XGBoost** 和 **LSTM** 神经网络分别建模。  
   - 根据验证集上的表现动态分配权重，加权得到最终预测。  
   - 支持缺失库自动跳过（如无TensorFlow则跳过LSTM）。

2. **自动化数据获取**  
   - 通过天天基金API获取基金历史净值（稳定接口），失败时自动降级为JS解析。  
   - 从新浪财经获取黄金ETF（518880）成交量作为替代量能指标。  
   - 自动处理SSL警告，适应网络环境。

3. **丰富的技术指标**  
   - 在原始价格和成交量基础上计算：移动平均线（MA）、指数平均（EMA）、RSI、MACD、布林带、ATR、CCI、OBV、动量、波动率、价格位置等。  
   - 这些指标作为特征输入模型。

4. **回测与绩效评估**  
   - 滚动窗口回测，支持定期重新训练模型（可配置频率）。  
   - 计算方向准确率、策略累计收益、年化收益、夏普比率、最大回撤、盈亏比等关键指标。  
   - 可选绘制净值曲线对比图。

5. **自动更新实际涨跌幅**  
   - 每次运行脚本时，自动从最新历史净值中获取已过交易日的实际涨跌幅，填充到预测日志中，确保统计准确。

6. **预测记录与统计**  
   - 所有预测结果保存在 `predictions_log.csv`，包含预测日期、集成预测值、各模型单独预测值、实际涨跌幅（自动更新）。  
   - 提供历史预测准确率、MAE、RMSE等统计。

### 🛠️ 使用方法

#### 环境要求

  ```bash
  pip install numpy pandas scipy matplotlib requests scikit-learn xgboost tensorflow
  ```
  （XGBoost和TensorFlow为可选，缺失则对应模型不启用）

#### 运行脚本
```bash

```
运行后出现菜单：
- **1. 预测明日涨跌幅**：获取最新数据，训练模型，输出次日预测。
- **2. 查看历史预测统计**：基于已有记录显示准确率、误差等。
- **3. 更新实际涨跌幅（手动）**：手动输入某日实际值（一般由自动更新替代）。
- **4. 运行回测**：使用历史数据模拟策略表现，生成回测报告。

### 📂 文件说明
- `predictions_log.csv`：每日预测记录（自动生成）。
- `backtest_result.csv`：回测明细（运行回测后生成）。
- `model_weights.json`：暂未使用，预留保存模型权重（可扩展）。

### ⚠️ 注意事项
- 数据来源于公开网络，接口可能变动，若获取失败请稍后重试。
- 预测仅作为量化研究参考，不构成投资建议，市场有风险。
- 首次运行会获取3000天历史数据（约10年），耗时数秒至数十秒，请耐心等待。
- 若网络连接问题导致API失败，脚本会自动尝试备用的JS解析方式。

### 📈 策略逻辑（回测中）
- 每日预测次日涨跌幅。
- 若预测上涨（>0），则次日开盘买入，收盘卖出，获得当日实际收益率。
- 若预测下跌或持平，则空仓，收益率为0。
- 以此模拟“只做多”的择时策略，并与“买入持有”基准对比。

### 🔧 自定义配置
在脚本开头的**配置参数**区域可修改：
- `FUND_CODE`：基金代码（默认002611）。
- `ETF_VOLUME_CODE`：用于成交量的ETF代码（默认518880）。
- `HISTORY_DAYS`：获取历史数据天数。
- `WINDOW_SIZE`：特征窗口大小。
- `TEST_RATIO`、`RETRAIN_FREQ`：回测训练集比例与重新训练频率。
